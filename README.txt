# Decrypt Network Inspector (Chrome DevTools Extension)

A custom Chrome DevTools extension that captures **XHR / Fetch** requests and allows you to:

- Inspect network requests similar to Chrome's Network tab  
- View **Headers, Payload, Response, and Params**
- Decrypt encrypted payloads/params/responses safely using your backend  
- Toggle between RAW and DECRYPTED view  
- Expand results in a JSON tree viewer  
- Debug encrypted APIs without exposing encryption keys
